# AI Agent
## AI Agent
스스로 의사 결정하며, 도구를 사용하여, 목표를 달성하는 시스템
- 도구 사용이나 다단계 추론을 활용
- LLM을 중심으로 다양한 외부 도구를 연결하여 구축
> **문제를 스스로 해결하는 시스템**

### 에이전트의 구조
사용자의 입력을 받아 응답을 생성
- 필요 시
    1. 정보를 검색하거나 (Retrieval)
    2. 툴을 사용하거나 (Tool)
    3. 메모리를 읽고 쓰며 (Memory)
    4. 결과를 종합해서 응답을 생성

## LangGraph
에이전트 간의 협업을 그래프 기반으로 설계하는 프레임 워크
- 그래프 : 에이전트들이 특정 목표를 위해 수행하는 일련의 과정, 워크플로우를 가리킴

### 주요 구성 요소
**1. 노드**
    - 그래프에서 특정 작업이나 판단을 수행하는 단위 
**2. 엣지**
    - 노드간의 연결을 나타내며 작업의 흐름을 정의
**3. 조건부 엣지**
    - 특정 조건에 따라 노드 간의 분기 처리를 가능하게 함
**4. 스테이트**
    - 그래프의 현재 상태 값을 저장 및 전달하는 역할

### 기본 그래프
1. 간단한 그래프
    - 입력, 처리, 출력으로 이어지는 단순 흐름
2. 라우팅
    - 특정 입력에 따라 다른 경로 선택
3. 리플렉션 
    - 추론과정을 다시 생각하여 자기 피드백을 생성하는 메커니즘
