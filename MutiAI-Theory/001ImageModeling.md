# Image Modeling
## 컴퓨터 비전이란?
- 사람이 사물을 인식하듯이, 컴퓨터가 이미지를 분석하고 이해하는 기술
- 카메라, 센서 등을 통해 입력된 이미지 데이터를 해석하여 의미 있는 정보를 추출

### 컴퓨터의 이미지 인식 과정
**1. 이미지 입력**
    - 픽셀 단위로 이미지 수집
    - 색상, 밝기 등의 기본 정보로 변환
**2. 전처리**
    - 노이즈 제거
    - 크기 조정
    - 정규화
**3. 특징 추출**
    - 모서리, 패턴, 질감 등을 추출
    - 고전적으로는 알고리즘, 딥러닝으로는 cnn
**4. 분류 및 인식**
    - 딥러닝 기반의 모델로 사물을 분석하고 분류
**5. 결과 출력**
    - 예측된 결과를 확률과 함께 출력
    - 객체인식 또는 분할로도 활용 가능

## CNN
### 컨셉
- 부분의 특징을 추출하는 여러 필터를 동시에 학습해 내어 만들고, 그걸로 찾는다
    **아마 병령처리의 신경망을 사용해서 그런 듯**

### 구조 및 처리과정
1. input_shape : 분석 단위이인 이미지 한장의 크기 (픽셀사이즈, 채널)
2. convolutional Layer(합성곱 레이어) : 필터로 지역적인 특성을 뽑는 과정
3. max pooling Layer : 뽑은 특징을 요약 (압축, 중요한 특징만 남기기)
4. Flatten : 어떤 형식의 데이터를 전부 1차원 데이터로 펼치기
