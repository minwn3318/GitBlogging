# 로그인 인증

## 로그인(인증)에 필요한 요소
1. 인증을 위한 정보(예: 아이디와 패스워드)
2. 인증을 했다는 증명(session 값, 토큰)
<- session 은 서버에 저장되는 공간(보통 특정 유저들의 인증 정보)
<- session 은 백엔드와 프론트가 한서버에 있지 않으면 사용이 불가

-> next.js 는 현재 백엔드와 서버가 다르므로 session 사용이 불가능 하다.
