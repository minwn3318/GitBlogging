# 컴퓨팅
## 컴퓨팅 진화
1. 가상화
- 서버 가상화

2. 컨테이너화
- 경량화, 이식성

3. 서버리스
- 응답과 요청시 짧고 비번할 때 유용
- 서버가 항시작동이 아니라 잠깐하는 응답과 요청에만 해당

4. 서버리스 컨테이너화
5. aws 맞춤 제작 및 특화

## 인스턴스
EC2 인스턴스는 아마존 클라우드 환경에서 사용되는 가상 컴퓨터
EC2 인스턴스는 전 세계 aws 리전의 물리적 서버에 호스트됨

### 최신 세대 인스턴스 유형의 이점
- 향상된 성능
- 비용 효율성
- 확장성

### 이름 및 태그
- 리로스를 관리, 검색, 필터링 가능

## Amazon Machine Image
이미지 - 인스턴스 탬플릿 이미지

### AMI 구성요소
- 인스턴스 볼륨
- 시작권한
- 블록 디바이스 매핑

### 이점
- 반복가능
- 재사용 가능
- 복구 가능
