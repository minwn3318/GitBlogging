## 판다스 - 데이터프레임 탐색
### 기본 데이터 정보
1. 기본정보 읽기
```
df.info()
```


2. 상세설명 읽기
```
df.describe()
```


3. 데이터 셋 크기 (행, 열)
```
df.size
```


4. 데이터 열(칼럼)
```
df.columns
```


5. 데이터의 값 종류 갯수 (값이 많은 순으로 나열)
- 데이터 프레임 뿐만 아니라 칼럼 리스트에도 해당됨
```
# 기본방법
df.value_counts()


# 비율적으로 나타내는 방법
df.value_counts(normalize='True')
```


6. 데이터 값 나열
- 데이터프레임 뿐만 아니라 칼럼 리스트에도 해당됨
```
df.values
```


7. 데이터 타입
- 데이터프레임 뿐만 아니라 칼럼 리스트에도 해당됨
```
df.dtypes
```


### 데이터 수치 정보
1. 평균값
- 수치 데이터만
```
df.mean()
```

2. 중앙값
- 수치 데이터만
```
df.median()
```


3. 최빈값
```
df.mode()
```


### 데이터 상태 정보
1. 수치형 조회

```
df.is_numberic()
```


2. 결측치 파악
```
# 결측치가 있는가
df.isna()


# 결측치가 없는가
df.notna()
```

## 판다스 - 데이터프레임 조회
1. 기본 조회
```
# 기본형
df.loc[행,열]


# 열 전용
df[열]
df.loc[:,열]

# 다중 열 검색
df.loc[:,[칼럼, 칼럼]]


# 범위로 조회
df.loc[시작번호:끝번호, 시작칼럼:끝칼럼]


# 조건 조회 (행에만 해당)
df.loc[(df[조건칼럼] 조건식, ....),열]

```

## 판다스 - 데이터프레임 변경
### 데이터 관련
1. 결측치 채우기
```
df[].fillna()
```

2. 값 바꾸기
```
# 기본
df.loc[행,열] = 값


# 매핑시켜 전부 바꾸기 (map, 안 맞으면 nan)
df[] = df[].map({:,:})


# 매핑시켜 전부 바꾸기 (replace, 미싱인 것도 그대로 둠)
df[] = df[].replace({:,:})
```


3. 타입 바꾸기
```
df[].astype()
```


4. 행 버리기
```
df.drop(행 혹은 columns=열명, inplace= 대체)
```
