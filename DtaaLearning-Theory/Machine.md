# 머신러닝 모델링
## 머신러닝
- 명령 없이도 스스로 데이터로 규칙과 패턴을 찾아내고 학습하는 기술
- 새로운 데이터가 주어졌을 때 예측하거나 분류할 수 있게 됨
- 머신이 데이터를 통해 배우는 방식

---
## 분류와 회귀
### 분류(Classification)
- : 예, 아니오를 판정하는 것
### 회귀(Regression)
- : 원리를 분석해 수치로 나타내는 것

분류와 회귀에 따라 알고리즘이 달라짐

---
## 모델, 모델링
### 모델
- 데이터로부터 패턴을 찾아 수학식으로 정리해 높는 것
### 모델링
- 오차가 적은 모델을만드는 과정

### 모델의 목적
**샘플을 가지고 전체를 추정**
- 샘플 = 표본
- 전체 = 모집단
- 추정 = 예측

### 종속변수
- 영향을 받아 종속적으로 변하는 값

## 평균과 오차
### 평균
- 통계학에서 가장 단순한 모델 중 하나 

### 오차 (이탈도)
- 실젯값과 모델 예측값의 차이

---
## 가변수화
### 가변수화
- 범주형 데이터를 독립된 열로 변환하는 과정
    - 점주형 문자열 데이터는 숫자로 변화되어야하기 때문
    - 변수화 과정에서 이진적으로 분류되기에 One-Hot-Encoding이라 함 

### get_dummies()
- 가변수화 시키는 함수
```
dumn_cols = ['smoke', 'day', 'time']
tip = pd.get_dummies(tip, columns=dumn_cols, drop_first=Treu, dtype=int)
```
- columns 옵션에 가변수화 대상 열을 리스트 형태로 전달하면 일괄 처리됨
- 열 이름이 새로 생기는 열 접미사로 사용됨
- drop=True를 지정해 가변수화 결과로 만들어진 열 중 하나르 제거해 자우공선성 문제를 예방함
- 가변수화 대상이던 원본열을 자동을 제거됨

## 데이터 분리
- 보통 데이터 셋을 학습용, 검증용, 평가용으로 분리함
**관측값과 목표값으로 나눠서 학습하고 평가 시에도 관측값과 목표값이 학습에서와 같게 나오기 위해 함**

---
## 과대적합 vs 과소적합
### 과대적합 (오버피팅)
- 학습 데이터에 대해서는 성능이 매우좋은데, 평가 데이터에 대해서는 성능이 매우 좋지 않은 경우
**학습 데이터에 대해서만 잘 맞는 모델**

### 과소적합(언더피팅)
- 학습 데이터보다 평가 데이터에 대한 성능이 매우 좋거나 모든 데이터에 대한 성능이 매우 안 좋은 경우
**모델이 너무 단순하여 적절히 훈련되지 않음**
