# 모델 성능 평가

---
## 회귀 모델 평가
- 회귀 모델이 정확한 값을 예측하기는 어려움
    - 예측 값이 실제 값에 가까울 수록 좋은 모델이라 할 수 있음
**예측한 값과 실제 값의 차이로 모델 성능 평가**

## 분류 모델 평가
- 분류 모델은 0인지 1인지를 예측하는 것
    - 예측 값이 실제 값과 많이 같을 수록 좋은 모델이라 할 수 있음
**정확히 예측한 비율로 모델 성능을 평가**

---
## 실젯값, 평균값, 예측값
### 실젯값
- 우리가 실제로 예측하고 싶은 값
    - 오리가 관심을 갖는 오차는 이 값과 예측값의 차이

### 평균값
- 이미 알고 있는 이미 존재하는 있는 평균으로 예측한 값
    - 우리 모델의 예측값이 평균값보다 오차를 얼마나 더 줄였는지 궁금

### 예측값
- 우리 모델로 새롭게 예측한 값
    - 우리 모델이 평균값보다 얼마나 잘 예측했는지 궁금

---
## 하나의 값
모델은 실젯값과 예측값의 차이인 오차의 크기로 평가함
**오차평균 합(SSE / y - y')** : 이 오차를 표현하는데 나열하기보단 합해진 값이 더 편함

### 오차 제곱의 합
**MSE**
**RMSE**

### 오차 절대값의 합
**MAE**
**MAPE**

### 결정계수

sst = (실제값 - 평균) 제곱 합
sse = 실제값 - 예측값 제곱 합
ssr = 평균 - 예측값 제곱합
sst = sse + ssr
r^2 = ssr / sst = 1 - sse / sst

## 평가한 값에 이름을 부여
**정확도**
**정밀도** : 정확함만 함 (성공한것만 이야기함)
**재현율** : 정확하다 믿고 밀고나감 (실패해도 계속 긍정으로 함)
**f1-score** : 조화평균

- 정밀도와 재현율은 서로 상반된 형태
