# 딥러닝 기초
## 패턴
- 데이터에 대한 일정한 규칙
- 패턴이 없는 데이터라면 노이즈

## 모델
- 데이터에서 패턴을 찾아 수학식으로 표현한 것
**모델링** : 가능한한 정확도 오차가 적은 모델을 만드는 과정
**모델 튜닝** : 가능한한 재현율 오차가 적은 모델을 만드는 과정

### 모델의 목적
- 샘플을 바탕으로 전체를 추정하기 위함
    - 샘플 : 표본, 일부, 과거의 데이터
    - 전체 : 모집단, 전체 집합, 현재와 미래의 데이터
    - 추정 : 예측, 추론

# 딥러닝 특징
- **정의** : 인공신경망을 이용한 머신러닝의 한 분야
- **특징** : 특징을 스스로 학습
- **데이터 요구량** : 대량의 데이터가 필요
- **처리 방식** : 다층 신경망 사용
- **계산 지원** : 강력한 GPU 등 고성능 자원이 필요
- **대상 데이터** : 비구조화 데이터 중심
- **대표 사례** : 이미지 인식, 자연어 처리, 음성 인식

## 특징 추출
- 모델이 중요한 특징을 스스로 학습해서 추출함

## 스케일링
- 수치최적화를 기반으로 하기 때문에, 입력 데이터의 스케일을 맞추는 것은 정확하고 빠른 학습, 안정적인 수렴, 효율적인 역잔파를 위해 필수적인 전처리 과정임

## 컴파일
- 모델이 학습을 시작할 수 있도록 손실함수, 최적화 방법, 평가 지표 등을 설정하는 단계
