# 자바 변수 타입 역할과 종류
## 변수타입
### 기본형(원시타입)
- 정수, 실수, 문자, 논리
- 오버 언더 플로우 발생
- 스택 메모리에 저장
- 실제 값이 쌓임

### 참조형
- 실제값 주소를 힙 메모리에 저장
- 배열, 클래스, 인터페이스, 열거형
- 얘는 그냥 쌓이는 게 아니라 객체로 쌓임

---
## 스택과 메모리 차이
- 스택 : 간단하게 빠르게
- 힙 : 복잡한걸 처리, 가비지 컬렉션하고 할당과 해제가 유동적

---
## 변수타입 소개
- 정수형 : 양수, 음수, 0 표현
- 실수형 : 부호, 지수부와 가수부로 구분되어 메모리에 저장되며 정수형 타이보다 표현 범위가 넓음
    - 해당 내용은 정밀도, 반올림 적합성에서 정밀도를 해야할 때 사용됨
- 문자형 : 똑같이 숫자가 들어가는 데, 매핑되는 문자표에 따라 문자가 나옴 (옛날 : 아스키코드, 현재 : 유니코드) 
- 논리형 : 참, 거짓 표현

---
## 변수의 특징
**변수**
- 다양한 정보를 저장할 수 있는 메모리를 지정하는 이름
- 명시적 데이터 타입을 선언해야만 함

**변수 선언만 하고 초기화 안했다**
- 원시타입 : 0
- 레퍼런스타입 : 널

**식별자**
- .으로 클래스안의 변수, 메서드를 구분함
